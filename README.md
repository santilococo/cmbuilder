# cmbuilder

This repository contains macOS apps optimized for ARM-based Apple Silicon. To easily install these apps, you can download the corresponding `*.dmg` file from the releases section or utilize my Homebrew tap:

For instance, to install the optimized version of mpv for Apple Silicon, execute the following commands:

```bash
brew tap santilococo/cmtap
brew install --cask santilococo/cmtap/mpv
```
